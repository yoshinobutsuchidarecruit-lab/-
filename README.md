## 概要
未経験からエンジニアを目指す中で、Webの基本構造を理解するために作成した自己紹介ページです。

## 使用技術
- HTML
- CSS
- JavaScript

## 実装内容
- HTMLでページ構造を作成
- CSSで読みやすさを意識したデザイン調整
- JavaScriptでボタンクリック時の表示切り替えを実装

## 工夫した点
- header / main / section を使い、構造を意識した
- JavaScriptはHTML読み込み後に実行されるよう、scriptタグの位置を工夫した

## 今後の改善点
- デザインの改善
- フォーム入力機能の追加
